# Anti-Corruption Court System

## 1) Public filing / sponsorship
- Plaintiff drafts claim
- Public sponsors review claim
- 1,000–10,000 sponsors commit dollars to file claim
- If threshold is reached, claim is filed and escrow is funded
- If threshold is not reached, claim does not proceed

## 2) Intake and admissibility
- Identity / KYC / fraud checks
- Duplicate / collision review
- Scope check: same defendant, conduct, and time window?
- Screening for frivolous or insufficient claims
- If rejected, case ends
- If accepted, case enters court flow

## 3) Case classification and control
- Case tier assigned
- Priority / expedite decision
- Assignment rule selected
- Scheduling and staffing decisions made

## 4) Pretrial / investigation flow
- Complaint screening
- Preliminary investigation
- Motions / evidence handling
- Conflict-of-interest checks at each stage
- Judges for these stages are assigned from the standing pool under stage-rotation rules

## 5) Trial panel selection
- Eligible pool formed from standing judge pool
- 15 judges randomly selected from the eligible pool
- No judge from prior stages of the same case can sit on trial panel
- Publicly auditable random selection / lottery

## 6) Trial, deliberation, and verdict
- 15 randomly selected judges review evidence
- Judges vote on criminal liability
- Judges vote on sentence
- If 80%+ agree that conduct is criminal and agree on sentence, sentence is enforced
- Otherwise case is acquitted, dismissed, retried, or appealed depending on rules

## 7) Enforcement and publication
- Sentence enforced
- Public finding published
- Sanctions / penalties / remedies executed
- Docket updated

## 8) Appeals and oversight
- Appeal panel reviews if appeal is triggered
- Peer review, audit, and bias checks may occur
- Final outcome recorded

## 9) Learning / feedback loops
- Outcomes update judge credibility estimates
- Outcomes update appeal-risk estimates
- Outcomes update staffing / priority / assignment policies
- Simulation and offline evaluation test policy changes before deployment

## Important structure
- Main case flow is linear: sponsorship -> intake -> classification -> pretrial -> trial -> verdict -> enforcement
- Feedback loops go only from later outcomes back into policy / credibility / simulation layers
- Case assignment and review do NOT feed back into the initial public sponsorship stage
